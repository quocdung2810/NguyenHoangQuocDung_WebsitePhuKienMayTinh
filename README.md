## Cấu trúc thư mục

- index.html: Trang chủ
- products.html: Danh sách sản phẩm
- product-detail.html: Chi tiết sản phẩm mẫu
- cart.html: Giỏ hàng mẫu
- contact.html: Liên hệ
- labs.html: Trang tổng hợp Lab thực hành
- labs/: Thư mục chứa từng bài Lab
- css/style.css: Giao diện responsive
- js/main.js: Tương tác menu, sản phẩm, giỏ hàng mẫu
- data/products.json: Dữ liệu sản phẩm mẫu
- assets/images/products: Ảnh sản phẩm
- docs: Tài liệu, ghi chú, báo cáo

## Cách chạy

Mở file index.html bằng trình duyệt hoặc dùng Live Server trong VS Code.
