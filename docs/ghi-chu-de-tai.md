# Ghi chú đề tài

## Chức năng nên có

- Trang chủ giới thiệu website
- Danh sách sản phẩm
- Tìm kiếm sản phẩm bằng JavaScript
- Trang chi tiết sản phẩm
- Giỏ hàng demo
- Form liên hệ
- Giao diện responsive cho điện thoại

## Việc cần làm tiếp

- Thêm ảnh thật cho sản phẩm trong assets/images/products
- Hoàn thiện giỏ hàng bằng localStorage
- Thêm đăng nhập/đăng ký nếu giáo viên yêu cầu
- Kiểm tra giao diện trên điện thoại
- Nén source code thành .zip trước khi nộp
